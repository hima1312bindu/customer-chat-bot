**Customer Support Chatbot (Transformer-Based)**
This project implements a transformer-based customer support chatbot that handles common e-commerce queries such as order tracking, cancellations, and returns. It uses a fine-tuned DistilBERT model for intent detection and a pretrained emotion detection model to understand user sentiment. By maintaining session memory and applying escalation logic, the chatbot delivers accurate, context-aware, and emotion-aware responses, improving customer experience and support efficiency.
